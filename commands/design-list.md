Call `design_list` to retrieve all projects.

Format the result as a markdown table with the following columns:

| Name | Last updated | Preview URL |
|------|-------------|-------------|

If no projects exist, tell the user there are no projects yet and they can create one with `/design <name> <prompt>`.
